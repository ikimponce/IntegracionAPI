Pasos Para la Ejecucion:

1-Instalar dependencias con Composer, ejecutando el siguiente comando 'composer install'
2-Ajustar la conexion a la base de datos segun corresponda en el .env(puede utilizar .env.example como plantilla)
3-Ejecutar las migraciones de la base de datos con el comando 'php artisan migrate'
4-Iniciar el servidor con el comando 'php artisan serve'

Consultar los endpoints en POSTMAN, se pueden importar desde el archivo proporcionado.
